# Machine Learning Lab – Logistic Regression Project Solutions

## Overview
This project applies Logistic Regression to predict whether a user will click on an online advertisement based on demographic and behavioral features.

The goal is to understand which factors (e.g., time spent online, age, income) influence ad-click behavior and help businesses improve targeted advertising strategies.

## Dataset Description

The dataset contains user information related to online advertisement behavior including:
- Daily Time Spent on Site
- Age
- Area Income
- Daily Internet Usage
- Gender (Male)
- Clicked on Ad (Target)
The objective is to predict whether a user clicks on an advertisement (classification problem).
